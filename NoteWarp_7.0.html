Словари содержат сокращения и их подстановки.

        
        
Добавление нового словаря

        
Создайте текстовый файл .txt с содержимым {}, переименуйте подходящим именем и перетащите в зону импорта.

        
        
Добавление сокращения

        
Выберите словарь для редактирования в левой панели. Введите сокращение и подстановку в правой панели, нажмите Сохранить.

		
        
Добавление меток быстрого перехода

        
Расставьте в шаблоне символы * в местах ввода значений. Если вам часто нужно редактировать, удалять или дополнять определённые разделы текста, заключите их в [1> такие нумерованные метки <1].
		
		
		
Добавление ссылок

        
 [1> Текст <1] можно дублировать сразу в нескольких местах - расставьте ссылки [[1]]  на соответствующий нумерованный раздел. 

        
		
Динамические теги даты и времени

        
Используйте их в подстановке, чтобы вставить текущие дату и время:

        

          
            
ТегПример значения

          
          
            
[дата]14.02.2026

            
[дата_iso]2026-02-14

            
[время]14:30

            
[время_iso]14:30:22

          
        

        
        
Конфликты сокращений

        
Если одно сокращение есть в нескольких активных словарях, оно не будет работать. Статус конфликтов отображается жёлтой полосой.

      

    


        
    

      

        
Модули расширяют функциональность редактора.

        
Установка модуля

        
Перетащите .js файл в зону импорта на вкладке «Модули».

        
Структура модуля

        
const MyModule = {
  id: "my-module",
  name: "Название модуля",
  version: "1.0",
  description: "Что делает модуль",
  process(text, cursorPos, api) {
    // Обработка текста
    return { text, cursorPos };
  }
};
for (let key in MyModule) exports[key] = MyModule[key];
