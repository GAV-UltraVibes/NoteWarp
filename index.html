Словари содержат сокращения и их подстановки.

        
Добавление нового словаря

        
Создайте текстовый файл .txt с содержимым {}, переименуйте подходящим именем и перетащите в зону импорта.

        
Добавление сокращения

        
Выберите словарь для редактирования в левой панели. Введите сокращение и подстановку в правой панели, нажмите Сохранить.

        
Добавление меток быстрого перехода

        
Расставьте в шаблоне символы * в местах ввода значений. Если вам часто нужно редактировать, удалять или дополнять определённые разделы текста, заключите их в [1> такие нумерованные метки <1].

        
Добавление ссылок

        
 [1> Текст <1] можно дублировать сразу в нескольких местах - расставьте ссылки [[1]] на соответствующий нумерованный раздел. 

        
Динамические теги даты и времени

        
Используйте их в подстановке, чтобы вставить текущие дату и время:

        

          
ТегПример значения

          
            
[дата]14.02.2026

            
[дата_iso]2026-02-14

            
[время]14:30

            
[время_iso]14:30:22

          
        

        
Конфликты сокращений

        
Если одно сокращение есть в нескольких активных словарях, оно не будет работать. Статус конфликтов отображается жёлтой полосой.

      

    

    
    
    
    

      

        
🧩 Возможности модулей

        

          
Модули расширяют функциональность редактора. Они могут:

          
    
            
  • 📝 Обрабатывать текст — автоматически изменять содержимое при вводе или по команде
    • 
            
  • 📚 Управлять словарями — добавлять сокращения и автозамены для специфичных задач
    • 
            
  • ⌨️ Добавлять горячие клавиши — свои комбинации для быстрых действий
    • 
            
  • 🔧 Создавать кнопки — на панели инструментов для вызова функций модуля
    • 
            
  • 💬 Показывать подсказки — выводить сообщения пользователю
    • 
          

        

        

          
Установка модуля

          
Перетащите .js файл в зону импорта на вкладке «Модули». Модуль появится в списке — включите его галочкой.

        

        

          
Управление модулями

          
    
            
  • ✅ Включение/выключение — галочкой в списке, без удаления
    • 
            
  • 📦 Экспорт — кнопка «Экспорт» сохранит модуль как .js файл
    • 
            
  • ✏️ Редактирование — выберите модуль, измените код в правой панели, нажмите «Сохранить»
    • 
            
  • 🗑️ Удаление — кнопка «Удалить» уберёт модуль полностью
    • 
          

        

        

          
Где брать модули?

          
Модули можно найти в репозитории проекта на GitHub или создать самостоятельно. Для создания модуля достаточно базовых знаний JavaScript.
